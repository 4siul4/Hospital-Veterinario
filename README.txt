Para ejecutar y probar el proyecto, hay que ejecutar la clase Execute, dentro de las carpetas: Hospital\src\main\java\com\Hospital

Los test unitarios se encuentran en la carpeta Hospital\src\test\java\com\Hospital

También se ha adjuntado la colección de postman con las llamadas y ejemplos de estas.
Se ha incluido una base de datos vacía en formato H2, ya que esta se informará ejecutando en orden la colección de Postman, la cual se puede visualizar desde un navegador en el siguiente enlace una vez la aplicación esté lanzada: http://localhost:8080/h2-console.
El usuario para acceder a la base de datos es: sa
Y la contraseña: password

Por último, se ha adjuntado la documentación de las llamadas en formato OpenApi/Swagger en otro archivo dentro de la carpeta principal en formato JSON.
Para visualizarlo se puede hacer desde la siguiente página web: https://editor.swagger.io.
